# Dr. Joshi Dental Care — Website

Premium single-page dental website. White + deep navy. No build step required.

## Quick Deploy (GitHub Pages)
1. Upload this folder to a new GitHub repo
2. Settings → Pages → Branch: main / folder: root → Save
3. Live in ~1 minute at `https://<your-username>.github.io/<repo-name>/`

## 5 Things to Update Before Going Live

| What | Where | Find |
|---|---|---|
| Phone number | `index.html` + `assets/js/main.js` | `9999999999` (replace all) |
| Doctor's full name | `index.html` | `Dr. [Name] Joshi` |
| Clinic address | `index.html` | `123, Main Road` |
| Email | `index.html` | `hello@drjoshidental.com` |
| Google Maps link | `index.html` | `maps.google.com/?q=Dr+Joshi+Dental+Care` |

## Stack
- Plain HTML + CSS + Vanilla JS (no build step, no npm)
- Google Fonts: Inter + Playfair Display
- Images: Unsplash (free, embedded URLs)
- Works on `file://` or any static host
